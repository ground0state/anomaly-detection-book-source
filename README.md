## 書籍について

本書は、Python を使用した異常検知について解説した本です。主に疎構造学習と密度比推定法について概要を説明し、スクラッチのサンプルコードを計算しています。サンプルコードはこのリポジトリに格納されています。

書籍のページは[こちら](https://techbookfest.org/product/6309354181492736)。
